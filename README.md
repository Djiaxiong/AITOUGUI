# AITOUGUI

Unity project for AI-assisted UGUI generation workflows and examples.

## Project Structure

- `Assets/AI_UGUI_Creator/` - core tooling and documentation
- `Assets/Examples/` - generated and sample UI cases
- `Packages/` - Unity package manifest
- `ProjectSettings/` - Unity project settings

## Notes

- This repository intentionally excludes Unity cache and local machine files.
- See `Assets/AI_UGUI_Creator/README.md` for module-level documentation.
