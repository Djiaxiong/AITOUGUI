# WindinatorLite

`WindinatorLite` is a curated subset extracted for AIToUGUI.

Included scope:
- SDF graphics base
- Rectangle, polygon, line, and layered canvas drawing
- Runtime shape drawers and required shaders
- Minimal `GenericPool<T>` dependency for shape batching

Excluded scope:
- LayoutBuilder
- Demo content
- Audio, palette, editor tooling outside the required runtime path
- Window/framework-level runtime and pooling systems

This folder is intentionally not a full copy of Windinator.
